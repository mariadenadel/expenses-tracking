# Expenses Tracking

Expenses Tracking is a web site for tracking your expense

## Installation

```bash
git clone https://github.com/mariadenadel/expenses-tracking.git
cd expenses-tracking
bin/rails db:migrate
bundler install
npm install
bin/rails server
```
